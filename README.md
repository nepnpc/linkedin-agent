# LinkedIn Agent

Autonomous LinkedIn content poster. Zero manual intervention.

A GitHub Actions cron wakes the agent, it reads its own post history so it never repeats, writes a new post using Groq Llama 3, and ships it. Untouched for weeks. Still going.

## Stack
- Python
- Groq Llama 3
- GitHub Actions
